You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply?num1=3&num2=4 -- multiply two numbers
  • /github?query=users/{username}/repos -- return GitHub user and list of all their repositories
  • /countries?name=france&capital=paris -- Fetch weather data for a given city and display
  • /weather?city=Phoenix,NewYork,LosAngeles&unit=metric --- Fetch data about independent countries
